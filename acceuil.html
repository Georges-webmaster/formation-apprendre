L'histoire des langages

Selon les systemes d'exploitation, il y a un langage approprié, le java est utilisé pour Windows et Mac.

Pour chaque système, il y un langage de programmation.

Pour ce qui est du web, il y a eu l'apparition des navigateurs.

En effet, les différents entreprises du web ont créés leurs navigateurs allant d'IE pour Microsoft, Mozilla pour la fondation Firefox, Safari pour Apple et Chrome pour Google maintenant Alphabet.

Il s'avère aussi que des langages de programmation ont été créés entre le HTML, le CSS, le JS, le PHP, SQL et Python.

Pour ce qui est des mobiles, il existe deux langages de programmation, Android pour Google et ios pour Apple.

Ce sont principalement les langages JAVA pour Android et Objective C pour ios

Il s'est ensuite avéré qu'il était utile d'utiliser un langage adapté pour les différents supports ou plus précisémment les terminaux.

C'est donc le cross Platform qui est apparu en 2011, c'est un modèle qualifé d'hybride.

Comprendre et fonctionnement de base d'une page web

Nous commençons avec le "Front" et sa spécificité de l'utilisation HTML et CSS3, le JavaScript.

C'est aussi les autres langages pour le back entre le PHP et ses variables, les bases de données, le SQL.

En effet, il est utile de comprendre que les pages web sont sotckés dans des serveurs.

Le langage SQL va alors être utilisé comme sous forme de collones et de rangées

Il était intéressant de comprendre le fonctionnement de la page web.

La page est affichée par le navigateur et elle est stockée dans un servveur.

Pour cela, nous devons charger une page depuis une serveur, il y a un chargement depuis le serveur avec la base de données. Nous pourrons alors obtenir une page web qualifiée de dynamique. En effet, l'utilisateur va pouvoir charger la page selon sa demande. Ce qui est en effet, ce qui caractérise par une page spécifique selon son identifiant et plus particulièrement selon la base de données qui va charger sa page.

La construction d'une page web

Il est utile de comprendre que le langage de programmation est un langage interprété.En effet, il va lire ligne par ligne le code qui est affiché par le navigateur. Nous utilisons des balises qui sont composées de chevrons.

Pour ce qui est de la structure d'une page web, nous devons tout d'abord déclarer la page afin qu'elle soit reconnue auprès du navigateur. Une page web est composée de plusieurs parties : l'entête avec la balise head et le corps de la page avec la balise body.

Nous pouvons dans la balise head produire plusieurs actions. Il nous est possible de :

    • Créer l'encodage pour tous les caractères spéciaux avec la balise meta charset UTF-8.
      Rendre adapatble l'affichage de la page web sur les différents terminaux avec La balise meta viewport
      Insérer un lien qui va associer les fichiers entre le ficher html et css. Nous pouvons faire ceci avec la balise Link et l'attribut.

    • Pour continuer, nous allons strtcurer notre page web avec des boîtes, celles-ci représentent en quelque les grands briques d'une page web. Nous les identifions avec une balise générique qui est div.
